Spring Boot + Flask AI Portfolio

건강검진 데이터 기반 비만 예측 서비스

키와 몸무게를 입력하면 Spring Boot가 Flask AI 서버로 예측 요청을 보내고, 머신러닝 모델의 비만 여부 결과를 웹 화면에 제공합니다.

AI 예측 시작 게시판 보기

Service Flow

  1. 사용자가 키와 몸무게 입력
  2. Spring Boot가 Flask API 호출
  3. AI 모델이 비만 여부 예측
  4. Thymeleaf 화면에 결과 출력

Main Features

포트폴리오 주요 기능

회원 / 로그인

Spring Security formLogin과 BCrypt 암호화를 적용한 회원 인증 기능입니다.

게시판

로그인 사용자만 접근 가능한 질문 게시판과 작성자 기준 수정/삭제 권한을 제공합니다.

AI 비만 예측

비회원도 사용할 수 있는 Flask AI 서버 연동 기반 예측 기능입니다.
