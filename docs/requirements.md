# 요구사항 정의서

## 프로젝트 목적

건강검진 데이터를 기반으로 사용자의 비만 여부를 예측하는 AI 웹 서비스를 개발한다.

Spring Boot와 Flask 서버를 분리하여 REST API 기반 구조를 학습하고, 머신러닝 모델 연동 경험을 목표로 한다.

---

## 사용자 요구사항

### 비회원 사용자

- 키와 몸무게 데이터를 입력할 수 있다.
- AI 예측 결과를 확인할 수 있다.

### 회원 사용자

- 회원가입 및 로그인이 가능하다.
- 게시판 기능을 사용할 수 있다.

---

## 시스템 요구사항

### Spring Boot 서버

- 사용자 요청 처리
- 화면 렌더링
- Flask 서버 API 호출
- 게시판 CRUD 처리
- Spring Security 인증 처리

### Flask AI 서버

- 머신러닝 모델 로드
- 입력 데이터 예측 처리
- 예측 결과 JSON 반환

---

## 예측 기능 요구사항

입력값:

- 키
- 몸무게

출력값:

- 비만 여부 예측 결과

---

## 비기능 요구사항

- 반응형 UI 지원
- REST API 기반 통신
- Bootstrap UI 적용
- GitHub 형상관리
